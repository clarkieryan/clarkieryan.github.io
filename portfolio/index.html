The Rainbow Venues

Working with Dontfret Media, we created an elegant solution for The Rainbow Venues. They wanted a completely self-sufficient event management system, we decided to use wordpress and then with heavy customisation we created post types for events and venues. The client, then wanted the site to link in with a ticket purchasing system. We also developed a photo gallery which pulls in their latest photos from Facebook and made the entire website mobile-friendly!

Smints

As part of a smaller project I had a need to use a static menu that locked onto the top of a page, for this I decided to use smint – a small JQuery plugin developed by Robert McCracken. However I felt it needed some extra functionality so I adapted it. I changed the plugin to add a class to the menu item that’s currently inside the active area. You can check out a demo here and the code is here.

Success! This code has been added to the official release – check it out here http://www.outyear.co.uk/smint/

1st Bedworth Scout Group

1st Bedworth was looking for a site that allowed them to easily update yet looked modern. To do this I worked with a number of members from the group to develop the solution that is currently in place. We used a responsive template to deliver the content with stylings similar to the scouting associations own branding. I developed a side bar calendar that linked up with the groups own Google calendar, and a photo gallery plugin where the parents and members can view photos taken from events the group hold.

Olicana Jewellery

Working with a designer, I developed an e-commerce system for Olicana Jewellery. To do this I used a mixture of WordPress and the Woo-Commerce plugin for wordpress, to deliver a basic e-commerce solution. Using the Woo-Commerce plugin I override the basic views to allow the layout the designer envisioned..
